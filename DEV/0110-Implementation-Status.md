# SnapPin 实现状态（对齐 PixPin 体验）

[PixPin link](https://pixpin.com/)

说明：
- **已完成**仅在你测试确认后标记。
- 其余均视为**待验证 / 进行中 / 未开始**。

## 已完成（用户已确认）
- 托盘右键菜单可用（右键弹出菜单）
- 截图进入“冻结帧”视觉暂停（含灰色蒙版 + 选区高亮）
- 选区截图位置准确（当前 150% 缩放、2560×1600）
- ESC 取消截图并退出截图模式（工具条消失）
- 截图预览拖拽稳定（无闪烁、无“幕布滑动”）
- PNG 保存可用（默认桌面路径）
- 截图后自动复制到剪贴板
- 截图后自动显示工具条
- 自动窗口吸附截图（无需模式切换：悬停自动高亮窗口，点击即截窗口；按住拖拽即区域截图）
- 截图后 `Ctrl+C` 等价于点击 `Copy`（复制并关闭当前工具条）

## 待验证（已实现，需你确认）
- 工具条顺序与动作入口（`Pin/Save/Copy/Mark/OCR/Close`）对齐 PixPin
  - 验证方式：截图后检查按钮顺序与点击响应；`Pin/Save/Copy` 语义与 PixPin 保持一致。
- Pin 基础可用（多实例）：截图 Pin、剪贴板 Pin、右键菜单 `Close/Destroy/Close All/Destroy All`
  - 验证方式：连续 Pin 多张图，分别测试右键菜单四个动作是否生效。
